# Student-Course-Management-System
A Python-based Student and Course Management System that allows educational institutions or instructors to manage students, courses, attendance, and grades efficiently through a simple CLI interface.

Features

👨‍🎓 Add Students with validation of IDs and email formats.

📚 Add Courses and enroll students.

📝 Mark Attendance for students per course, automatically recording dates.

🏆 Add Grades for students and calculate averages.

📊 Generate Full Reports showing:

Enrolled students

Individual grades and average

Attendance records and percentage

Highlights for outstanding students

✅ Interactive menu-driven CLI for easy usage.

🛡 Error handling and validation to prevent incorrect data entry.

Technologies Used

Python 3

datetime for date handling

re for email validation
